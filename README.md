# Filemaker tinyMCE

realtime html generation + pdf export

tested only on latest filemaker version 19.4.2.204

**default language is german, change it in `src/editor.js`**

you will need [nodejs](https://nodejs.org/en/) to build it

1. npm install

    ```bash
    npm i
    ```
    or

    ```bash
    yarn
    ```
2. build
    ```bash
    npm run build
    ```
    or

    ```bash
    yarn build
    ```
